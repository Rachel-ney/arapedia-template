# Matrice Parcel

Base du build Parcel comprenant : 
- autoprefixer
- postcss
- reset-css
- sass
- fontawesome

## Pré-requis 
Node.js et npm

## Start 
Lancer un `npm install`

**Watch** : npm run dev

**Build** : npm run prod

**Attention ! Dans le .postcssrc si modules est à true le nom des classes CSS sera modifié et donc plus viable avec le css en place !**
